# API 仕様書（PromptGrade）

## 概要

このドキュメントは、プロンプト評価アプリ「PromptGrade」で使用する FastAPI バックエンドの API 仕様を定義します。

---

## 📌 共通情報

- ベース URL：`http://localhost:8000`（開発時）
- データ形式：JSON
- 認証：なし（初期 MVP 段階）

---

## ✅ エンドポイント一覧

| メソッド | パス        | 機能概要                                                     |
| -------- | ----------- | ------------------------------------------------------------ |
| POST     | `/evaluate` | ユーザーが入力したプロンプトを評価し、スコアと改善提案を返す |

---

## ▶️ POST `/evaluate`

### 📖 機能概要

ユーザーから送られたプロンプトを OpenAI API に渡し、その評価結果と改善案を返します。

---

### 🔸 リクエスト

#### ヘッダー

```http
Content-Type: application/json
```

#### ボディ

```json
{
  "prompt": "プロンプト文字列"
}
```

| フィールド名 | 型     | 必須 | 説明                                     |
| ------------ | ------ | ---- | ---------------------------------------- |
| `prompt`     | string | 必須 | ユーザーが評価してほしいプロンプト文字列 |

---

### 🔹 レスポンス

```json
{
  "score": 8,
  "feedback": "このプロンプトは明確で目的もはっきりしていますが、もう少し具体性があると良いでしょう。",
  "suggested_prompt": "より具体的な条件を追加した改善済みプロンプト"
}
```

| フィールド名       | 型      | 説明                               |
| ------------------ | ------- | ---------------------------------- |
| `score`            | integer | 評価スコア（1〜10）                |
| `feedback`         | string  | 良い点・改善点などのフィードバック |
| `suggested_prompt` | string  | 改善されたプロンプトの例           |

---

## 🔧 内部処理の流れ

1. `/evaluate` に対してプロンプトを含むリクエストを受け取る
2. OpenAI API の `chat/completions` を呼び出し
   - モデル：`gpt-4` または `gpt-3.5-turbo`
   - システムプロンプトで評価と提案を指示
3. OpenAI のレスポンスを解析し、評価点・フィードバック・改善案を抽出
4. フロントエンドへ JSON 形式で返却

---

## 💡 使用モデルと API パラメータ

| パラメータ        | 値                              |
| ----------------- | ------------------------------- |
| モデル            | `gpt-4`（必要に応じて切替可能） |
| temperature       | 0.7                             |
| max_tokens        | 1024                            |
| top_p             | 1.0                             |
| frequency_penalty | 0                               |
| presence_penalty  | 0                               |

## 🚫 エラーレスポンス例

### 1. プロンプト未入力

```json
{
  "detail": "Prompt is required."
}
```

ステータスコード：`400 Bad Request`

---

### 2. OpenAI API 接続エラー

```json
{
  "detail": "Failed to get response from OpenAI API."
}
```

ステータスコード：`500 Internal Server Error`

## 🛠 今後の拡張予定（スコープ外）

- OpenAI API キーの動的設定（ユーザー入力）
- 評価項目のカスタマイズ（明確さ / 効果性など）
- モデル選択 UI との連携（gpt-3.5 / gpt-4）

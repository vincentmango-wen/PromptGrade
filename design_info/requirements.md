# 要件定義書（プロンプト評価アプリ）

## 1. アプリ名（仮）

**PromptGrade（プロンプトグレード）**

---

## 2. アプリの目的

- ユーザーが入力した **プロンプト（LLM への指示文）** に対して、AI が「評価」と「改善提案」を返す。
- 初心者〜中級者向けプロンプト作成支援ツールとして、プロンプト設計力の向上を支援する。

---

## 3. 対象ユーザー

- ChatGPT や Claude などの LLM を使い始めた初心者〜中級者
- プロンプトエンジニアリングを学びたい開発者・学生
- 社内の AI 活用を進めたいビジネス担当者

---

## 4. アプリの機能（MVP で実装する範囲）

| 機能名                   | 概要                                                    | 実装優先度 |
| ------------------------ | ------------------------------------------------------- | ---------- |
| プロンプト入力フォーム   | ユーザーが任意のプロンプトを入力できる UI               | 高         |
| プロンプト評価           | OpenAI API を使ってプロンプトの内容を分析・スコアリング | 高         |
| 改善提案生成             | 評価に基づいた改善点と修正プロンプトを生成              | 高         |
| 履歴保存（ローカル）     | 入力と評価結果をブラウザのローカルに保存（後で見る用）  | 中         |
| 使用モデルの切替（任意） | GPT-3.5 と GPT-4 など選べる機能                         | 低         |

---

## 5. 使用技術

| 項目           | 技術スタック                                       |
| -------------- | -------------------------------------------------- |
| バックエンド   | FastAPI (Python)                                   |
| フロントエンド | HTML/CSS + JavaScript（簡易 UI）または軽量な React |
| API 連携       | OpenAI API（`chat/completions`）                   |
| モデル         | GPT-3.5 または GPT-4                               |
| デプロイ方法   | ローカル or Render/Heroku（無料枠）                |
| データ保存     | 初期は無し（履歴はローカルストレージ）             |

---

## 6. 非機能要件

| 項目           | 内容                                                   |
| -------------- | ------------------------------------------------------ |
| セキュリティ   | API キーは `.env` に保管。Git 管理外にする             |
| レスポンス時間 | 最大 5 秒以内を目指す                                  |
| UI/UX          | PC/スマホ両対応のレスポンシブデザイン（後回しでも OK） |

---

## 7. 開発体制

- 開発者：1 名（プログラミング学習歴 1 年）
- 想定期間：2〜4 週間（学習兼ねて進行）
- コメント付きで理解しながら進める

---

## 8. スコープ外（今は実装しない）

- ログイン・ユーザー管理
- プロンプトの分類タグ
- API 使用量の制限・制御
- 評価アルゴリズムの独自実装（OpenAI 任せ）

---

## 9. 想定ユーザーストーリー（例）

> ユーザーがアプリを開いて、書いたプロンプトを貼り付けると、数秒後に「このプロンプトの良い点」「悪い点」「改善案」が表示される。

---

## 10. 想定する使用例

- ChatGPT で営業メールを作るためのプロンプトが本当に効果的か確認したい
- 社内研修で「良いプロンプトとは？」を教える時に見本として使いたい

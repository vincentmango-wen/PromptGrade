# 実装計画書（PromptGrade）

## 📌 目的

このドキュメントは、プロンプト評価アプリ「PromptGrade」の開発タスクを機能単位で整理・管理し、学習者 1 人でも迷わず進められるようにすることを目的としています。

---

## ✅ タスク一覧（MVP スコープ）

| タスク番号 | 機能                 | 内容                                           | 優先度 | ステータス |
| ---------- | -------------------- | ---------------------------------------------- | ------ | ---------- |
| T01        | 環境構築             | Python 仮想環境 + FastAPI セットアップ         | ★★★    | 未着手     |
| T02        | `.env`管理           | OpenAI API キーを`.env`に保存し読み込む        | ★★★    | 未着手     |
| T03        | `main.py`作成        | FastAPI アプリの起動エントリーポイント実装     | ★★★    | 未着手     |
| T04        | `/evaluate`ルート    | API の POST ルートを追加、リクエストを受け取る | ★★★    | 未着手     |
| T05        | スキーマ定義         | `schemas/prompt_schema.py`で pydantic 定義     | ★★★    | 未着手     |
| T06        | OpenAI 連携          | `openai_service.py`で API 呼び出し処理実装     | ★★★    | 未着手     |
| T07        | レスポンス解析       | `parser.py`で score 等を正規表現で抽出         | ★★★    | 未着手     |
| T08        | レスポンス返却       | JSON 形式で結果を返す処理                      | ★★★    | 未着手     |
| T09        | フロント仮 UI        | HTML フォーム + JS で API へ POST              | ★★☆    | 未着手     |
| T10        | 結果表示処理         | JS でレスポンスを受け取り画面に表示            | ★★☆    | 未着手     |
| T11        | ローカル履歴保存     | localStorage に履歴保存（任意）                | ★☆☆    | 未着手     |

| T15        | コードにコメント追加 | すべてのコードに初心者向けの解説コメント       | ★★★    | 未着手     |

## 🗂 タスク進行メモ

- 優先度記号：★（低）〜★★★（高）
- ステータス：`未着手` / `進行中` / `完了`
- タスクを順に実行し、学習と並行して進行
- 1 ファイルごとにコミットして、変更を追いやすくする
